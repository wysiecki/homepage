Jak używać

1. Wpisz wyrażenie — Wpisz standardowe 5-polowe wyrażenie cron (minuta, godzina, dzień miesiąca, miesiąc, dzień tygodnia).

2. Lub wybierz szablon — Kliknij dowolny przycisk szablonu, aby natychmiast wypełnić popularne harmonogramy.

3. Przeczytaj opis — Narzędzie tłumaczy wyrażenie na zrozumiały język, abyś mógł zweryfikować, czy robi to, czego oczekujesz.

4. Sprawdź czasy uruchomienia — Następne 5 zaplanowanych uruchomień jest obliczanych na podstawie bieżącej daty i godziny.

Referencja pól cron:

*    *    *    *    *
|    |    |    |    |
|    |    |    |    +-- Dzień tygodnia  (0-7, Niedz=0 lub 7)
|    |    |    +------- Miesiąc         (1-12)
|    |    +------------ Dzień miesiąca  (1-31)
|    +----------------- Godzina         (0-23)
+---------------------- Minuta          (0-59)

Znaki specjalne: * (dowolny) , (lista) - (zakres) / (krok)
