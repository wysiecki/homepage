JSON Formatter

Formatuj, waliduj i upiększaj dane JSON. Działa w całości w Twojej przeglądarce.

Nieprawidłowy JSON

Jak używać

Wklej lub wpisz swój JSON w pole wejściowe po lewej stronie. Narzędzie waliduje JSON w czasie rzeczywistym i pokazuje wszelkie błędy składni z numerami linii i kolumn.

  1. Formatuj — Formatuje JSON z wybranym wcięciem (2 lub 4 spacje).
  2. Minifikuj — Kompresuje JSON do jednej linii, usuwając wszystkie białe znaki.
  3. Kopiuj — Kopiuje sformatowane wyjście do schowka.
  4. Wyczyść — Resetuje oba pola wejścia i wyjścia.

Wszystkie operacje wykonywane są po stronie klienta w przeglądarce. Twoje dane nigdy nie opuszczają Twojego urządzenia.
