Anleitung

          

            

              1. JWT Token einfügen — Fügen
              Sie Ihren Token in das Eingabefeld oben ein. Der Decoder akzeptiert Tokens im
              Standard-Format
              header.payload.signature.
            

            

              2. Dekodierte Ausgabe prüfen. Der
              Header zeigt den Algorithmus und Token-Typ. Der Payload enthält Claims wie
              Aussteller, Betreff, Zielgruppe und Ablaufzeit.
            

            

              3. Token-Gültigkeit prüfen.
              Wenn der Token einen
              exp-Claim hat, zeigt der Decoder an, ob
              er abgelaufen, gültig oder noch nicht gültig ist. Zeitstempel werden in
              menschenlesbare Daten umgewandelt.
            

            

              4. Einzelne Abschnitte kopieren — Nutzen
              Sie die Kopieren-Buttons neben jedem dekodierten Abschnitt.
            

            

              Dieses Tool verifiziert keine JWT-Signaturen — es dekodiert nur den
              Base64url-kodierten Inhalt. Keine Daten verlassen Ihren Browser.
