Regex Tester

Reguläre Ausdrücke mit Live-Match-Highlighting, Capture Groups und Flag-Unterstützung testen.

/ /
Keine Treffer
Geben Sie oben ein Muster und eine Testzeichenkette ein

Anleitung

1. Muster eingeben — Geben Sie einen regulären Ausdruck in das Musterfeld ein. Keine Begrenzer nötig, nur das reine Muster.

2. Flags setzen — Aktivieren Sie global (g), Groß-/Kleinschreibung ignorieren (i), mehrzeilig (m) oder dotAll (s) Flags über die Checkboxen.

3. Testtext hinzufügen — Fügen Sie Ihren Teststring ein oder tippen Sie ihn. Treffer werden in Echtzeit beim Tippen hervorgehoben.

4. Treffer prüfen — Der Treffer-Details-Bereich zeigt jeden Treffer mit vollständigem Text, Capture Groups und Zeichenindex.
