Anleitung

1. Ausdruck eingeben — Geben Sie einen Standard-5-Feld-Cron-Ausdruck ein (Minute, Stunde, Tag des Monats, Monat, Wochentag).

2. Oder Vorlage wählen — Klicken Sie auf eine Vorlage, um gängige Zeitpläne sofort einzufüllen.

3. Beschreibung lesen — Das Tool übersetzt den Ausdruck in Klartext, damit Sie überprüfen können, ob er das Gewünschte tut.

4. Ausführungszeiten prüfen — Die nächsten 5 geplanten Ausführungen werden ab dem aktuellen Datum und der aktuellen Uhrzeit berechnet.

Cron-Feld-Referenz:

*    *    *    *    *
|    |    |    |    |
|    |    |    |    +-- Wochentag    (0-7, So=0 oder 7)
|    |    |    +------- Monat        (1-12)
|    |    +------------ Tag im Monat (1-31)
|    +----------------- Stunde       (0-23)
+---------------------- Minute       (0-59)

Sonderzeichen: * (beliebig) , (Liste) - (Bereich) / (Schritt)
